Phaser game develop boilerplate
-------------------------------

## how to use

1. git clone the project
2. npm in 
3. node devServer.js
4. open browser <http://127.0.0.1:4000>
5. write your game , auto fresh in browser
6. use gulp to build your game (in `build` folder )
